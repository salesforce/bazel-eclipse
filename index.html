Bazel Eclipse Installation

This is the installation site for Bazel Eclipse. You can read up on Bazel Eclipse in the documentation

There is also an update site for installing Bazel Eclipse. Follow these instructions:

  • In Eclipse, select the menu Eclipse < Help < Install New Software...
  • Click the Add button
  • Enter a name, 'Bazel Eclipse' for example
  • For location, enter https://opensource.salesforce.com/bazel-eclipse/latest/
  • Click the Add button
  • Check the Bazel Eclipse Feature
  • Click on Next until the feature is installed
